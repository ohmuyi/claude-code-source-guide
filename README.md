# 《Claude Code源码精讲》

![Claude Code源码精讲封面](./cc.png)

这是一本面向工程师的 Claude Code 源码导读。它不按文件清单做流水账，而是按系统边界来组织内容：启动、回合、工具、权限、TUI、任务、MCP、远程会话、记忆、技能、插件与上下文压缩。

这套文档的目标很明确：帮助读者建立稳定的源码心智模型，而不是在海量 TypeScript 文件里迷路。

## 这本书适合谁

- 想系统理解 Claude Code 内部结构的开发者
- 想研究 agent runtime、CLI、TUI、权限系统与工具编排的人
- 想把源码阅读结果沉淀成可复用方法的人

## 内容结构

全书分为四篇：

1. 启动与执行世界观  
从入口、回合推进和 Tool 协议开始，建立整体运行模型。

2. 任务、权限与执行内核  
解释 Task、权限决策链和 `tool_use` 的真实落地过程。

3. TUI、状态与后台任务  
分析终端运行时、状态模型、任务面板和可观察性机制。

4. 协议接入、生态与长期运行  
覆盖 MCP、远程会话、记忆、技能、插件与 compact。
